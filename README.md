A simple and interactive two-player Tic-Tac-Toe game built using Python. This project runs in the terminal and allows players to take turns, track scores, and replay multiple rounds.

FEATURES:
->🧑‍🤝‍🧑 Two-player mode (Player X and Player O)
->🎯 Turn-based gameplay with input validation
->🧠 Win detection using all possible winning combinations
->📊 Live scoreboard (Player 1 wins, Player 2 wins, and ties)
->🔁 Option to replay multiple games
->❌ Handles invalid inputs and occupied positions gracefully
